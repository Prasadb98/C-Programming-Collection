💻 C Programming Projects Collection

A collection of beginner-to-intermediate level C programming projects. This repository demonstrates core programming concepts like loops, conditionals, functions, arrays, and basic problem-solving techniques.

---

📌 Projects Included

⚡ 1. Electricity Bill Calculator

A program to calculate electricity bills based on units consumed using slab rates.

🚀 Features

- Accepts user input for units
- Calculates bill using slab system
- Displays total payable amount

🧮 Sample Logic

Units Range| Rate per Unit
0–100| ₹5
101–200| ₹7
200+| ₹10

---

🎓 2. Grade Management System

A program to calculate student grades based on marks.

🚀 Features

- Input marks of subjects
- Calculate total and average
- Assign grade based on performance

🧮 Grading Criteria

Marks (%)| Grade
90+| A+
80–89| A
70–79| B
60–69| C
<60| Fail

---

📚 3. Library Book Management System

A basic system to manage library records.

🚀 Features

- Add new books
- View all books
- Search book by ID/name
- Delete book records

---

➕ 4. Sum of N Numbers

A simple program to calculate the sum of first N natural numbers or user-entered numbers.

🚀 Features

- Accepts value of N
- Calculates sum using loop
- Displays result

🧮 Formula Used

- Sum of first N natural numbers = N × (N + 1) / 2

▶️ How to Run the Projects

🔧 Prerequisites

- GCC Compiler installed
- Terminal / Command Prompt

⚙️ Steps

1. Clone the repository:
   
   git clone https://github.com/your-username/c-programming-projects.git
cd c-programming-projects

2. Compile any project:
   
   gcc filename.c -o output

3. Run the program:
   
   ./output
